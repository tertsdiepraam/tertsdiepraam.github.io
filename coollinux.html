Cool Linux Things

A list of useful and cool linux utilities and tricks (in no particular order).

KDE Connect / GSConnect

Connect your phone to your KDE or Gnome desktop. It can:

  • share notifications
  • transfer files
  • transfer browser tabs
  • control your mouse and keyboard from your phone
  • control media both ways
  • control slideshows
  • run custom commands

Flatpak

Flatpak is the best and easiest way to install a lot of apps. I recommend distro's that support Flatpak out of the box (e.g. Pop!_OS). A lot of commercial software, such as Spotify, is not available through apt, but is available through Flatpak. Install Flatseal to manage the permissions of Flatpak apps. Sidenote: Snap is not bad, but Flatpak is much better.

Flatpak Guide | Flathub

Proton

You can install Steam on Linux too. Look up how to use Proton to run a lot of games that don't support Linux natively. Want to know whether a game will work well via Proton? Look the game up on ProtonDB.

Alternatives to Windows Software

A lot of commercial software, such as Microsoft Office and Adobe Creative Cloud, is sadly not available on Linux. Luckily there are plenty of alternatives. A few are listed here. One thing to note is that these are not always drop-in replacements, many work slightly differently and take some getting used to. However, they don't lack much in functionality once yoiu get used to them.

  • Microsoft Word → LibreOffice Writer
  • Microsoft PowerPoint → LibreOffice Impress
  • Microsoft Excel → LibreOffice Calc
  • Abobe Illustrator → Inkscape
  • Abobe Premiere → DaVinci Resolve (not open source), KDEnlive (open source)
  • Adobe AfterEffects → DaVinci Resolve (fusion, not open source), Blender
  • Adobe Photoshop → GIMP (photo manipulation), Krita (drawing)
  • Adobe Reader → Okular or Evince
  • Autodesk Maya → Blender

For more alternatives, check AlternativeTo.

Command Line

There's no way around it: if you're using Linux, you have to use the command line sometimes. In most distributions, the standard shell is Bash, which is pretty good. Look up some tutorials to familiarize yourself with the most common commands:

  • ls: list files in directory
  • mkdir: make a new directory
  • cd: change directory
  • mv: move files
  • cp: copy files
  • rm: remove files
  • cat: show contents of a file and join input from multiple files
  • less: show content of a file with a nicer interface than cat
  • pwd: show present working directory
  • apt: install, update and remove software (on Debian derivatives, such as Ubuntu)

Bash Guide for Beginners

There are some optional programs which can be really helpful, or just fun:

  • bat: a better cat/less
  • ripgrep: search files really fucking fast
  • exa: a better ls
  • nnn: a super fast command line file manager

If you are fed up with Bash, try other shells! Here are a few examples:

  • Zsh (basically Bash with extra features)
  • Fish (very user friendly)
  • Nushell (more experimental, but really cool)

Password Manager

Everyone should be using a password manager, no matter what operating system you're on. When I chose my password manager I chose it based on the following criteria:

  1. must be well-established
  2. must be open source
  3. must have clients on a major platforms (Linux, Windows, Mac OSX, Android, iOS, Firefox & Chrome)

The first two criteria are for security reasons, I want to be able to check whether it is secure, or, if I don't check myself, I want other's to have check it's security. The last criterium is just convenience, as I want it to work on all platforms I use currently and I might use in the future. All this brought me to Bitwarden, essentially the best (open source) password manager around. And it has a free hosted plan too (in contrast with KeePass)!

I installed the Bitwarden Linux, Firefox and Android clients and was ready to go.
